# План продвижения и развертывания Lerne (Phase 3)

Этот документ описывает технические шаги по запуску лендинга и интеграции рекламных инструментов в приложение Wordy.

## 1. Хостинг (Развертывание сайта)
Для размещения лендинга выбрана платформа **GitHub Pages**.
- **URL**: `https://[username].github.io/Anki-card-andder/landing/` (после настройки в репозитории).
- **Преимущества**: Бесплатно, автоматическое обновление при пуше в основную ветку, привязка к открытому коду.

## 2. Интеграция "Крючков" в приложение
Чтобы превратить Wordy в эффективный лид-магнит, необходимо внедрить следующие механизмы:

### А. Водяные знаки в карточках (Card Watermarks)
- **Файл**: `api/anki_api.py`
- **Суть**: Добавление небольшого, стильного текстового блока или ссылки в подвал (Footer) каждой создаваемой карточки.
- **Текст**: *"Generated by [Lerne Assistant]([URL_Сайта])"*

### Б. Рекламный блок в интерфейсе (UI Promo)
- **Файл**: `ui/main_window.py`
- **Суть**: Размещение кнопки или кликабельной ссылки в нижней части окна.
- **Текст**: *"✨ Попробуйте Lerne: Будущее изучения языков"*
- **Действие**: Открытие сайта в браузере при клике.

## 3. Распространение (Дистрибуция)
- Обновление `README.md` на GitHub с ярким баннером Lerne.
- Подготовка 15-секундного демо для Shorts/TikTok.
- Пост в r/Anki с анонсом бесплатного инструмента и упоминанием Lerne.
